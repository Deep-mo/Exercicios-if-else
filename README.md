# Exercicios-if-else
Uma lista de exercicios no mesmo codigo, envolvendo diversos pontos do csharp... No mesmo codigo tu pode acessar 6 exercicios 
O tema dos 6 exercicios são : 
•Exercicio 1 : identifica se o usuario é maior de idade;
•exercicio 2 : identifica se o numero de input é par ou impar;
Exercicio 3 : contexto de ser usando para identificar se a nota(input) do usario seria alta o bastante para ser aprovada em uma prova;
•Exercicio 4 : compara entre numeros dados por input se eles são iguais ou qual é maior que o outro;
•Exercicio 5 : exercicio simples de login e senha ;
•Exercicio 6 : Correlaciona um numero com um dia da semana;
